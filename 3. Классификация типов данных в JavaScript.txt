Существует 8 типов данных, они делятся на 2 категории - простые типы(примитивы) и обьекты(комплексные).
Обьекты деляться еще на спец.обекты и обычные обьекты.
В js существует ограничение на большие числа в стандартом типе чисел, оно должно быть не больше чем 2 в 53 степени. Для этого есть другой тип данных BigInt  специально созданный для больших чисел.
Массив не является отдельным типом данных, это частный случай обьекта
Существуют специальные числовые значения которые относятся к обычному типу чисел это - infinity(также может быть отрицательным -infinity) мы можем это получить при делении на 0. Второй спец. числовое значение это NaN(not a number), мы можем получить его если проведем операцию которая не поддается математической логике.(допустим перемножить строку и число)

var age = null;
В JavaScript null не является «ссылкой на несуществующий объект» или «нулевым указателем», как в некоторых других языках. Это просто специальное значение, которое имеет смысл «ничего» или «значение неизвестно».
В частности, код выше говорит о том, что возраст age неизвестен.



Значение undefined, как и null, образует свой собственный тип, состоящий из одного этого значения. Оно имеет смысл «значение не присвоено».
Если переменная объявлена, но в неё ничего не записано, то её значение как раз и есть undefined:
var x;
alert( x ); // выведет "undefined"

В явном виде undefined обычно не присваивают, так как это противоречит его смыслу. Для записи в переменную «пустого» или «неизвестного» значения используется null.
